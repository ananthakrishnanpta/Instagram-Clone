# Instagram Clone

A responsive Instagram-inspired social media web application built using Bootstrap 5 with minimal custom CSS and JavaScript. This project recreates the core Instagram user interface, including feeds, stories, profiles, and messaging screens while maintaining a clean and mobile-friendly design.

## Features

### Authentication

* Login Page
* Registration Page
* Forgot Password Interface

### Home Feed

* Instagram-style posts
* User profile information
* Post captions
* Like, Comment, and Share actions
* Responsive feed layout

### Stories

* Horizontal scrolling stories section
* Circular profile thumbnails
* Responsive design

### User Profile

* Profile image
* Bio section
* Posts, Followers, and Following counts
* Responsive gallery grid

### Explore Page

* Masonry-style image grid
* Trending content layout
* Mobile and desktop optimized

### Messaging Interface

* Chat list
* Conversation panel
* Message input area

### Responsive Design

* Mobile-first implementation
* Tablet support
* Desktop support
* Bootstrap Grid System

## Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/instagram-clone.git
```

2. Navigate to the project directory

```bash
cd instagram-clone
```

3. Open `index.html` in your preferred browser.

No server setup or additional dependencies are required.
